# BAVYA Attendance Management System

A professional, high-performance Attendance Management System for biometric devices (ZKTeco).

## Features
- **Biometric Synchronization**: Real-time log fetching from ZKTeco devices.
- **Leave Management**: Comprehensive leave policy management with pro-rata calculations.
- **Employee Management**: Detailed employee registry with shift mapping.
- **Reporting**: Live attendance overview and detailed reporting.
- **Modern UI**: Built with FlatLaf Enterprise styling for a premium feel.

## Tech Stack
- **Language**: Java 17
- **Framework**: Swing (FlatLaf, MigLayout)
- **Database**: MySQL
- **Build Tool**: Maven

## Setup
1. Configure database in `bhspl_config.ini`.
2. Run using Maven: `.\mvnw.cmd compile exec:java`.
